# java-cli-gradle-jacoco-spock-test-hello-world

## Description
A POC for gradle app using spock framework.
Also dumps jacoco reports
to the same folder.

## Tech stack
- java
- gradle
  - spock
  - jacoco

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
